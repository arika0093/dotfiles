# dotfiles

## install
```sh
# install required package (if you wanted)
$ bash install-packages/console.sh
$ bash install-packages/fonts.sh
# some setting file copied
$ bash .bin/install.sh
# ...and restore vscode setting (if installed)
$ bash .bin/vscode_restore.sh
```
